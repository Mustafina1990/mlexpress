# ML Expresstäd AB - Professional Cleaning Services

A modern, responsive single-page application (SPA) for a professional cleaning company built with React, Vite, and Tailwind CSS.

## Features

- 🏠 **Modern Design**: Clean, professional interface with smooth animations
- 📱 **Responsive**: Fully responsive design that works on all devices
- ⚡ **Fast Performance**: Built with Vite for lightning-fast development and builds
- 🎨 **Beautiful UI**: Styled with Tailwind CSS and Heroicons
- 🔗 **Client-side Routing**: Smooth navigation with React Router
- 📞 **Contact Forms**: Easy-to-use contact and quote request forms
- 💼 **Service Showcase**: Detailed service offerings and pricing

## Pages

- **Home**: Hero section, services overview, testimonials, and call-to-action
- **Services**: Comprehensive cleaning services with pricing and descriptions
- **About**: Company story, team, values, and why choose us
- **Contact**: Contact information, service areas, and booking form

## Technologies Used

- React 19
- Vite
- Tailwind CSS 4
- React Router DOM
- Heroicons
- PostCSS
- ESLint

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

4. Preview production build:
   ```bash
   npm run preview
   ```
